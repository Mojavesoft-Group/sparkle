# Sparkle
![Sparkle logo](doc/logo.gif)

[License](LICENSE)

Sparkle is a modding system for Snap! and Jameson (with more languages on the way!). It's a fork of the [Crackle extension](https://github.com/CrackleTeam/CrackleSDK) designed to speed up development and improve the overall user experience.

More stuff coming soon!
